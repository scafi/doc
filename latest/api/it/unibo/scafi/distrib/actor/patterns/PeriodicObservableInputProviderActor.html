PeriodicObservableInputProviderActor

Related Doc: package patterns

abstract class PeriodicObservableInputProviderActor[K, V] extends Actor with PeriodicBehavior with ObservableActorBehavior

Abstract actor, which is an ObservableActorBehavior with PeriodicBehavior, that is intended to notify its observers with messages with a key-value format.

K

The type of the name.

V

The type of the value.
